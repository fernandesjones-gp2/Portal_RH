## 2. Como Preencher (Ordem Importa)

```
Ordem recomendada:

1. SEGMENTO DE CLIENTE  → Para quem? (persona primária)
2. PROBLEMA             → Qual a dor? (top 3)
3. PROPOSTA DE VALOR    → Como resume em 1 frase?
4. SOLUÇÃO              → Que features resolvem? (top 3 do MVP)
5. CANAIS               → Como chegar nos clientes?
6. ESTRUTURA DE CUSTOS  → Quanto custa operar?
7. FONTES DE RECEITA    → Como ganha dinheiro?
8. MÉTRICAS CHAVE       → Como sabe se está dando certo?
9. VANTAGEM INJUSTA     → O que é difícil de copiar?

Regras:
├── Não gastar mais de 20 minutos na primeira versão
├── Usar post-its mentais (cada item = máximo 3-4 bullets)
├── Revisar semanalmente com dados novos
├── Fazer 1 canvas por SEGMENTO (não 1 canvas geral)
└── Caixas vazias são OK — revelam onde falta clareza
```

## 4. Value Proposition Canvas

### Template

```
┌──────────────────────────────┐    ┌──────────────────────────────┐
│      SEU PRODUTO             │    │      SEU CLIENTE             │
│                              │    │                              │
│  Criadores de Ganho:         │◄──►│  Ganhos desejados:           │
│  [O que seu produto cria]    │    │  [O que o cliente quer]      │
│                              │    │                              │
│  Produtos & Serviços:        │◄──►│  Jobs-to-be-Done:            │
│  [O que você oferece]        │    │  [O que precisa fazer]       │
│                              │    │                              │
│  Alívio de Dores:            │◄──►│  Dores:                      │
│  [O que seu produto alivia]  │    │  [O que frustra/impede]      │
└──────────────────────────────┘    └──────────────────────────────┘
```

O FIT acontece quando cada Dor tem um Alívio, cada Job tem um Serviço,
e cada Ganho desejado tem um Criador de Ganho.

## 5. Análise Competitiva

### Template de mapa competitivo

```markdown
| Critério | NutriPlan (nós) | Dietbox | Planilha+WhatsApp | Nutrium |
|----------|----------------|---------|-------------------|---------|
| **Preço** | R$49/mês | R$180/mês | Grátis | R$99/mês |
| **Plano alimentar** | Templates rápidos | Completo | Manual | Completo |
| **App do paciente** | Sim (simples) | Sim | Não | Sim |
| **Financeiro** | Básico | Não | Planilha | Não |
| **Facilidade** | Muito fácil | Médio | Fácil (mas manual) | Médio |
| **Tempo de setup** | 5 min | 30 min | 0 (já usa) | 20 min |
| **Target** | Autônomas iniciantes | Clínicas | Qualquer | Clínicas |
```

### Perguntas a responder

```
Para cada concorrente:
├── Quanto cobram? (pricing model)
├── Quem é o público deles? (pode ser diferente do seu)
├── O que fazem BEM? (não subestime)
├── O que fazem MAL? (oportunidade)
├── O que os clientes reclamam? (reviews G2, App Store)
├── Qual o tamanho deles? (funcionários, funding, users estimados)
└── Estão crescendo ou estagnados?

Para o cenário:
├── Tem solução dominante? (winner takes all ou mercado fragmentado)
├── Quanto investimento seria necessário para entrar?
├── Qual a barreira de saída para clientes do concorrente? (lock-in)
└── Existe "não consumo"? (pessoas que deveriam usar mas não usam nada)
```

## 6. Posicionamento

### Template de posicionamento

```
Para [SEGMENTO]
que [SITUAÇÃO / NECESSIDADE],
[PRODUTO] é um [CATEGORIA]
que [BENEFÍCIO PRINCIPAL].
Diferente de [ALTERNATIVA PRINCIPAL],
nós [DIFERENCIAL COMPETITIVO].
```

### Exemplo

```
Para nutricionistas autônomas
que perdem horas em planilhas e WhatsApp,
NutriPlan é uma ferramenta de gestão de consultório
que reduz o tempo administrativo de 2 horas para 20 minutos por dia.
Diferente de Dietbox e Nutrium (caros e complexos),
nós focamos em simplicidade e preço acessível para profissionais solo.
```

### Regras de posicionamento

```
├── Ser específico (não "para todos os profissionais de saúde")
├── Ter um diferencial REAL (não "somos melhores")
├── O diferencial deve importar para o segmento
├── Ser honesto sobre o que NÃO faz (ajuda mais que atrapalha)
└── Posicionamento muda conforme o produto amadurece
```
